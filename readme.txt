This plugin helps to change WordPress Page Permalinks, which is by default enabled.

== Description ==

This plugin helps to change WordPress Page Permalinks on a WordPress site running 3.5 and beyond.
The permalinks can be changed manually.

WordPress page permalinks is not changable by default. This plugin helps to change WordPress Page Permalinks and this plugin provides an easy way to change it once for all.

== Installation ==

=== From within WordPress ===
1. Visit 'Plugins > Add New'
1. Search for 'Change WP Page Permalinks'
1. Activate Change WP Page Permalinks from your Plugins page.
1. Go to "after activation" below.

=== Manually ===
1. Upload the `change-wp-page-permalinks` folder to the `/wp-content/plugins/` directory
1. Activate the Change WP Page Permalinks plugin through the 'Plugins' menu in WordPress
1. Go to "after activation" below.

=== After activation ===
1. Enter your custom permalink and save changes
1. Go to Settings > Permalinks and resave your permalinks. This will flush the page permalink url.
1. Boom! You have succesfully change your page url.

To re-enable the default page permalinks, just deactivate the plugin through the 'Plugins' menu and resave your permalinks.

== Frequently Asked Questions ==

= How to change the page permalinks? =

Go to 'Settings > Change WP Page Permalinks' and enter your custom page permalink and hit 'save changes'.

= Is there an admin interface for this plugin? =

Yes. You can access this from 'Settings > Change WP Page Permalinks'

= How to check this is working? =

Open any page of your website and you will see the change.

The use the plugin to test it.
